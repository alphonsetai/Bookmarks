### Fedora 
```
# yum update
 
# yum install PyQt4

# yum install qt qt-demos qt-designer qt4 qt4-designer
```

### Ubuntu
```
# apt-get install python-qt4 pyqt4-dev-tools qt4-designer
```

```
yum whatprovides */pyuic4
yum install PyQt4-devel
```
